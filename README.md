<p align="left">
Hallo! Ik ben Joffrey Verschoor, welkom op mijn GitHub-profiel. Hier deel ik de projecten die ik heb ontwikkeld. Momenteel is mijn portfolio niet beschikbaar, omdat ik deze aan het updaten ben.
